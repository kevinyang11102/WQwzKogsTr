# 社区爱心养老管理系统 python1158

## 项目概述

社区爱心养老管理系统是基于前后端分离的架构设计，采用Python作为后端开发语言，结合Django框架和MySQL数据库，前端则使用Vue.js构建。该系统致力于为社区养老提供便捷的信息化管理，涵盖个人健康、图书借阅、活动参与等多个方面。

## 技术栈

- 后端：Python 3.7，Django框架
- 数据库：MySQL 5.7
- 前端：Vue.js

## 功能模块

### 用户功能

- 首页浏览
- 个人中心：个人信息更新、管理
- 健康信息管理
- 图书借阅与归还信息管理
- 活动信息查看与报名

### 管理员功能

- 系统首页管理
- 用户信息管理
- 健康信息管理
- 公共书籍管理
- 借阅与归还信息管理
- 图书分类与还书入库
- 活动信息与活动分类管理
- 系统管理

## 系统角色

- 用户：使用系统各项功能的普通角色
- 管理员：对系统进行管理与维护的高级角色

## 运行环境

- 后端：Python 3.7 环境搭建，Django框架配置
- 数据库：MySQL 5.7 或兼容版本
- 前端：Vue.js环境，推荐使用现代浏览器

## 部署步骤

1. 环境搭建：配置Python 3.7及Django，设置MySQL数据库
2. 数据库初始化：导入系统所需的SQL文件
3. 前端构建：使用Vue.js及相关依赖项编译静态文件
4. 部署应用：将编译后的前端文件与后端应用部署到服务器

## 目录结构

```
/backend              # 后端代码目录
  /apps               # Django应用目录
  /config             # 系统配置文件
  ...
/frontend             # 前端代码目录
  /src                # Vue.js源码目录
  ...
```

- 前后端分离架构，易于维护与扩展
- 基于Python的Django框架，开发效率高
- 系统涵盖社区养老的多个管理方面，功能全面
